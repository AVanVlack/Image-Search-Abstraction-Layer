Image Search Abstraction Layer

This api provides a abstraction of the flickr image search service. This project made for practice in node.js/express and building api's by Andrew Vanlack

Example Inputs

  1. https://vanvlack-imagesearch.herokuapp.com/search/cats?page=2
  2. https://vanvlack-imagesearch.herokuapp.com/history?page=1
